
# Análisis de Ventas en Fashion Retail
Este proyecto de data science analiza datos de ventas de una tienda de moda, abordando tareas clave de limpieza de datos y análisis exploratorio.

## Objetivo
Detectar y manejar valores faltantes y outliers, explorar tendencias en las ventas, y responder preguntas estratégicas que puedan guiar decisiones de negocio.

## Análisis Realizado
1. **Detección y manejo de valores faltantes y outliers**: Identificación y corrección de inconsistencias en los datos.

2. **Análisis exploratorio de datos (EDA)**:
* ¿Cuáles son los productos más vendidos?
* ¿Qué productos generan mayores ingresos?
* ¿Cómo varían las ventas en diferentes períodos de tiempo (diarias, semanales, mensuales, anuales)?
* ¿Existen patrones de estacionalidad en las ventas?

## Resultados
Este análisis brinda insights sobre el desempeño de los productos, comportamiento de los clientes y patrones de compra, ayudando a optimizar estrategias de marketing y gestión de inventarios.
